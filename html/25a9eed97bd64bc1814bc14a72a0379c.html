Step 2: Open the scilab-codes-analysis folder and then locate and open the folder Ramp_Analysis

Step 3: Copy the step test data file to the folder Kp-tau-order1.

Step 4: Change the Scilab working directory Ramp_Analysis folder.

Step 5: Open the file ramp_virtual.sce in scilab editor and enter the name of the data file (with extention) in the filename field.

Step 6: Save and run this code and obtain the plot
