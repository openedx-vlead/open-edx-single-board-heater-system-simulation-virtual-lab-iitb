Step 1: Download the Analysis folder from the simulator tab. The download will be in zip format. Extract the downloaded zip file. You will get a folder scilab-codes-analysis.

Step 2: Open the scilab-codes-analysis folder and then locate and open the folder Sine_Analysis .

Step 3: Copy the sine test data in to this folder.

Step 4: Change the Scilab working directory Sine_Analysis folder.

Step 5: Open the file sine-analysis.sce in scilab editor and enter the name of the data file (with extention) in the filename field.

Step 6: Put the value of frequency f for the calculation of amplitude ratio and phase di erence and execute it. Here f means input frequency.

Step 7: Expect the values of amplitude ratio and phase difference on the scilab console.
